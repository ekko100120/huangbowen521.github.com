JUnit annotation方式

JUnit中提供了一个expected的annotation来检查异常。

这种方式看起来要简洁多了，但是无法检查异常中的消息。

ExpectedException rule

JUnit7以后提供了一个叫做ExpectedException的Rule来实现对异常的测试。

这种方式既可以检查异常类型，也可以验证异常中的消息。
